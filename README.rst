Exploring Mandlebrot Plots
##########################

This is a C++/GLUT project for exploring Mandlebrot plots.

Clone this project for your own experimenting!

..  code-block:: shell

    $ git clone https://github.com/rblack42/Mandlebrot.git

To run:

..  code-block:: shell

    $ make
    $ ./demo (on Linux/Mac)
    $ demo (on PC)
